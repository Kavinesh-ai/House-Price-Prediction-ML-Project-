# House Price Prediction using Linear Regression

## Overview

This project predicts house prices using features such as area, bedrooms, and bathrooms with Linear Regression.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn

## Features Used

* Area
* Bedrooms
* Bathrooms

## Workflow

1. Load Dataset
2. Data Preprocessing
3. Train-Test Split
4. Linear Regression Model
5. Prediction and Evaluation

## Results

The model predicts house prices and is evaluated using the R² Score metric.
